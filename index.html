How to Use the Boston Matrix

To use this tool, follow these steps:

  1. Prepare a CSV file with the following columns:
    • Column A: Product name
    • Column B: Purchase price (ex VAT)
    • Column C: Sales price (inc VAT)
    • Column D: Quantity sold
  2. Upload the CSV file using the file input above.
  3. Click "Generate Boston Matrix" to visualize your data.

Ensure that the data in the CSV file is correctly formatted. You can download a template below.
